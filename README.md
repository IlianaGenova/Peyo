# Peyo
A student project for HackTUES 6

## Used hardware:
- Radio "Accord 104"
- Arduino Uno
- Raspberry Pi
- RGB led strip
- Amplifier
- Potentiometer

## Used software
- Python flask
- HTML&CSS
- C code(for microcontrollers)
- OrCAD Capture(for schematic)

## Webpage/Hardware functionalities
- Play Radio
- Play Spotify
- Play from USB
- Mute
- Control Volume
